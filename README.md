# ShopSnap

Your Social Shopping Platform.

## How to run

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Akankshavdixit/ShopSnap.git

2. **Install Dependencies**

`cd server
npm install`

`cd ../client
npm install`


3. **Run**
   
`cd server
nodemon index.js`

`cd ../client
npm run start`
